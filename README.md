### 1. 멀티모듈 & 아키텍쳐 

* movie-adpater : 외부로 부터 들어오는 요청 & 내부에서 외부로 나가는 응답을 처리하는 모듈  
* movie-application : 서비스의 비즈니스 로직을 처리할 수 있는 모듈
* movie-common : 공통으로 사용하는 코드를 관리하는 모듈
